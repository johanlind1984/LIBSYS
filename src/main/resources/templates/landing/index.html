Logga in.

Klicka här för att logga in i bibliotekssystemet. I bibliotekssystemet kan du låna böcker och e-böcker. Du kan även söka efter ett specifik seminarium eller lista alla kommande seminarium.

Skaffa bibliotekskort.

Klicka här kan du registrera dig och få ett bibliotekskort som du kan använda för att låna våra böcker eller e-böcker.

Sök.

Klicka här kan du söka efter böcker, e-böcker och seminarium som finns i biblioteket. Vi har böcker från hela världens hörn och kanter.

Varför digitalisering.

Klicka här för att läsa om varför vi har digitaliserat biblioteket.

Om oss.

Klicka här för att läsa lite om biblioteket, vår historia och hur vi uppstod.
